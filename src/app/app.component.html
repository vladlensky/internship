Стажировка:
Мы ищем в команду frontend разработчика!
Вам предстоит разработка фронтенд части разнообразных высоконагруженных портальных решений и не только.
Основные используемые технологии: JavaScript (ES5 / ES6 / ES7), TypeScript, SCSS, Webpack, Gulp, RxJS и другие.
Основные фреймворки: jQuery, AngularJS, Angular 5, React. Взаимодействие с бэкендом .NET / Java по средствам Rest API, WebSockets и нативных подходов как Razor, JSP.
Задачи обещают быть интересными!
Условия стажировки:
Мы берем стажера в свою команду, чтобы в будущем получить квалифицированного специалиста.
  • Хорошего знания JavaScript ES5.
  • Понимания основных принципов ООП и паттернов проектирования.
  • Владения навыками адаптивной, кросбраузерной верстки (HTML5, CSS3).
Со своей стороны, мы гарантируем:
  • Официальное трудоустройство на время стажировки.
  • Выполнение реальных проектных задач под руководством опытного наставника.
  • Полностью гибкий график, частичная занятость от 20 часов в неделю.
  • Гарантированное трудоустройство при успешном прохождении стажировки.
